Glossaire des balises HTML

<!DOCTYPE html>
Indique au navigateur qu il s agit d un document HTML5.
<html lang="en">
Balise racine de tout document HTML, avec l attribut lang qui spécifie la langue du contenu.
<head>
Contient des métadonnées et des informations sur le document, telles que le titre et les liens vers les ressources.
<body>
Contient le contenu visible de la page web, comme les titres, les paragraphes, les images, etc.
<h1>
Balise de titre de niveau 1, utilisée pour les titres principaux.
<p>
Balise de paragraphe, utilisée pour structurer le texte.
<hr>
Balise de ligne horizontale, utilisée pour séparer visuellement les sections du contenu.
<pre>
Balise de préformatage, utilisée pour afficher du texte avec une mise en forme préservée, souvent utilisée pour afficher du code source.
<dl>
Balise de liste de définition, utilisée pour créer une liste de termes et leurs définitions.
<dt>
Balise de terme de définition, utilisée pour définir un terme dans une liste de définition.
<dd>
Balise de description de définition, utilisée pour fournir la définition d un terme dans une liste de définition.
